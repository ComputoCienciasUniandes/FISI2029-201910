Filtro de señales y extracción de datos¶

De manera frecuente nos encontramos con que no tenemos acceso a los datos, es por esto que el siguiente ejercicio pretende mostrar una forma de extraer datos de ciertas bases de datos. para esto, revise la pagina, la idea es extraer los datos de las temperaturas, del oceano en "Northwest Arm, Halifax, Canada". Primero existen dos formas utiles de acceder a los datos.

Forma dificil¶

De ves en cuando los datos no estn como los quisieramos por cuestiones de seguridad aveces se encriptan o tal vez estn en cierto formato que pesa menos(binario por ejemplo).

Inicio = '20111118'
Final   = '20121125'
response=urllib.request.urlopen('http://lobo.satlantic.com/cgi-data/nph-data.cgi?min_date='+Inicio+'&max_date='+Final+'&y=temperature')
leer=response.read()
data = StringIO(leer.decode())
r = csv.DictReader(data,dialect=csv.Sniffer().sniff(data.read(1000)))
data.seek(0)
date, temp = [],[]
date, temp = zip(*[(datetime.strptime(x['date [AST]'], "%Y-%m-%d %H:%M:%S"), x['temperature [C]']) for x in r if x['temperature [C]'] is not None])
temp = np.array(temp)
temp = temp.astype(np.float)

Filtro¶

Primero se escogen los parametros que se desean para hacer el filtro entre estos est el orden del filtro y la frecuencia para hacer el filtro, de igual forma se puede efectuar un filtro de tipo pasabajos o pasa altos entre otros. El filtro a utilizar ac ser el de Butterworth, as de esta forma tenemos que:

In [10]:
N  = 2    # Orden del filtro
Wn = 0.01 # Corte de frecuancia
B, A = signal.butter(N, Wn)
In [11]:
temp_filtrada = signal.filtfilt(B,A, temp)

As de esta forma podemos filtrar el valor de temperaturas y luego graficar la seal original y la seal filtrada, al igual que el ruido como la diferencia entre la seal original y el ruido.
